# Notesy

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 16.1.0.

## Installation

```bash
# install node modules

$ yarn install
```

## Running the app
```bash
# build the project 
# build artifacts will be stored in the `dist/` directory

$ ng build
```

```bash
# run dev server 
# http://localhost:4200/

$ ng serve
```
